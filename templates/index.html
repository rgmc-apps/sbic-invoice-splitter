How it works

  1. Select a multi-page PDF containing multiple sales invoices.
  2. The file is uploaded directly to secure cloud storage — bypassing the 32 MB Cloud Run limit.
  3. Each page is split into its own PDF and the SI number is extracted automatically.
  4. Download a ZIP containing all individual PDFs, each named SI_{code}_{number}.pdf.

SI extraction (tried in order)

  • text-crop — text from your configured crop region (highest priority).
  • text-bottom — text from the bottom 40% of the page.
  • text-right — text from the right half.
  • text-full — full-page text.
  • fallback — page named page_NNNN.pdf if nothing found.

Notes

  • PDF only. No hard size limit — files upload directly to cloud storage.
  • Files are automatically losslessly optimized before processing.
  • Password-protected PDFs are not supported.
