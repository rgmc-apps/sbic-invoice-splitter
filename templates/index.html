How it works

  1. Upload a multi-page PDF containing multiple sales invoices.
  2. Each page is split into its own PDF file.
  3. The SI number is automatically read from the text under the barcode or near the lower-right corner of each page.
  4. Download a ZIP containing all individual PDFs, each named SI_{code}_{number}.pdf.

Extraction methods (tried in order)

  • text-bottom — fast text extraction from the bottom 40 % of the page.
  • text-right — text extraction from the right half of the page.
  • text-full — full-page text extraction.
  • barcode — renders the page as an image and decodes any barcode (Code 128, QR, etc.).
  • fallback — if no SI number is found, the page is named page_NNNN.pdf.

File requirements

  • PDF only  ·  Max {{ upload_limit_mb }} MB per upload
  • Files are automatically optimized before processing.
  • If your file exceeds the limit, use the Optimize button to compress it first.
  • Password-protected PDFs are not supported.
